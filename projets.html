02 · Projet scolaire — Trinôme

Gestion d'incidents — Application C#

BTS SIO · Avec Safouan & Ibrahim · 2024–2025

C# Windows Forms MySQL

Contexte

Application Windows Forms développée en C# permettant la gestion d'un parc informatique fictif : déclaration d'incidents, suivi des interventions, gestion du matériel et du personnel. Projet réalisé en trinôme avec Safouan et Ibrahim sur 4 séances, avec base de données MySQL et architecture orientée objet (classes Matériel, Intervention, Personnel).

Fonctionnalités

→ Interface technicien : gestion du matériel, prise en charge et résolution d'incidents

→ Base de données MySQL avec 5 tables : incident, intervention, matériel, personnel, utilisateur
→ Déploiement via setup généré par Visual Studio (Publish)

03 · Projet scolaire

Application Android — GSB RDV

BTS SIO · Android Studio · 2025–2026

Android Java SQLite

Contexte

Application mobile Android développée en Java dans le cadre du BTS SIO, inspirée du laboratoire GSB. Elle permet à un délégué médical d'enregistrer des professionnels de santé (pharmaciens, généralistes), de prendre des rendez-vous avec eux via un calendrier, d'afficher le planning d'une journée et de rechercher un professionnel par ville ou code postal.

Fonctionnalités

→ Enregistrement de professionnels de santé avec nom, type, adresse, mail, téléphone
→ Prise de RDV via CalendarView avec sélection du professionnel via Spinner
→ Affichage du planning d'une journée (liste des RDV par date)
→ Recherche de professionnels par ville ou code postal via requête SQLite

04 · Projet scolaire — Trinôme

Site Web GSB — PHP MVC & API REST

BTS SIO · Avec Ibrahim & Safouan · 23 mars – 2 avril 2026

PHP MVC API REST

Contexte

Site web vitrine pour le laboratoire pharmaceutique fictif GSB (Galaxy Swiss Bourdin), développé en PHP avec une architecture MVC2 et une API REST. Réalisé en trinôme sur 4 séances avec méthode Agile (Scrum). Le projet couvre la consultation de médicaments, l'inscription à des activités, la gestion de professionnels de santé et une interface d'administration sécurisée.

Fonctionnalités

→ Consultation des médicaments avec effets thérapeutiques, secondaires et interactions
→ Activités complémentaires GSB avec formulaire d'inscription
→ Interface admin sécurisée : CRUD professionnels de santé et planification des RDV
→ Mentions légales, conformité RGPD, script de sauvegarde de la base de données
